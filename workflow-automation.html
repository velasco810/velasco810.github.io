Understanding Automation Rules

ClickUp's automation engine lets you apply changes automatically based on triggers and conditions. When you set up an automation, the system will apply your specified actions whenever the trigger conditions are met.

Pro Tip: To apply changes across your workspace efficiently, use bulk automation rules. The system will apply changes to all matching items automatically, saving you hours of manual work.

How to Apply Automation Workflows

  1. 1

    Select Your Trigger

    Choose what event will trigger the automation. ClickUp will apply the workflow whenever this condition is detected.

  2. 2

    Define Your Conditions

    Set filters to control when the automation should apply. You can apply rules based on status, assignee, dates, or custom fields.

  3. 3

    Configure Actions to Apply

    Specify what changes to apply when triggered. The system can apply updates to multiple documents, tasks, or fields simultaneously.

  4. 4

    Apply to Your Workspace

    Save and activate. ClickUp will immediately begin to apply your automation across the selected scope.
